# 社交分享组件使用指南

## 📋 功能特性

**SocialShare.vue** 组件提供了完整的社交分享功能：

- ✅ **微信分享**：显示二维码，方便扫码分享
- ✅ **微博分享**：打开微博分享页面
- ✅ **QQ 分享**：打开 QQ 分享页面
- ✅ **复制链接**：一键复制当前页面链接

## 🚀 使用方法

### 基础用法

在任何日记页面中添加：

```vue
<SocialShare />
```

### 完整示例

```markdown
---
title: 今天的学习成果
date: 2026-03-12
tags: [学习，技术]
---

# 今天的学习成果

内容...

<SocialShare />
```

## 🎨 样式说明

### 按钮颜色

- **微信**：绿色渐变（#07c160 → #06ad56）
- **微博**：红色渐变（#e6162d → #cf1428）
- **QQ**：蓝色渐变（#12b7f5 → #0da8e0）
- **复制**：品牌色渐变

### 响应式

- **桌面端**：显示图标和文字标签
- **移动端**：只显示图标，节省空间

## 🔧 技术实现

### 组件特性

- 使用 Vue 3 Composition API
- 获取当前页面信息（`useData()`）
- 响应式设计，移动端和桌面端适配
- 流畅的动画效果

### 分享方式

**微信**：
- 点击显示二维码弹窗
- 用户扫码后可分享到朋友圈或好友

**微博/QQ**：
- 打开官方分享页面
- 自动填充当前页面链接和标题

**复制链接**：
- 使用 Clipboard API
- 降级方案兼容旧浏览器
- 2 秒后自动恢复按钮状态

## 📱 浏览器兼容性

### 必需支持

- ES6+ 语法
- Clipboard API（复制功能）
- CSS Grid 和 Flexbox

### 推荐浏览器

- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

## 🎯 注意事项

### 微信二维码

当前版本使用占位符显示，需要集成二维码生成库（如 qrcode.js）来生成真实的二维码。

### 分享链接

- 确保网站已部署到公网
- 本地开发时使用 `localhost`，功能受限

### 自定义配置

如需自定义分享平台或样式，可以修改 `SocialShare.vue` 文件。
