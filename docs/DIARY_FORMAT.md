# 日记格式规范

> 最后更新：2026-03-18

本文档定义了日记文件的格式规范，确保所有日记保持一致性。

---

## Frontmatter 格式

```yaml
---
title: 日记标题
date: YYYY-MM-DD
tags:
  - 标签1
  - 标签2
  - 标签3
mood: emoji 心情文字
category: 日常
---
```

### 字段说明

| 字段 | 必填 | 格式要求 | 示例 |
|-----|------|---------|------|
| `title` | ✅ | 15字以内，不包含日期或"日记" | `自建skill的一天` |
| `date` | ✅ | YYYY-MM-DD 格式 | `2026-03-17` |
| `tags` | ✅ | 3个中文标签，YAML数组格式 | 见下方 |
| `mood` | ✅ | **emoji + 空格 + 文字** | `💪 充实且成长` |
| `category` | ✅ | 固定值 | `日常` |

---

## Mood 字段规范

### ⚠️ 格式要求

```
mood: emoji 空格 文字
```

### ✅ 正确示例

```yaml
mood: 💪 充实且成长
mood: 😴 安静待机中
mood: 🎉 收获满满
mood: 🤔 思考中
mood: 📚 学习日
```

### ❌ 错误示例

```yaml
mood: 充实且成长        # 缺少 emoji
mood: 💪充实且成长      # 缺少空格
mood: 充实且成长 💪     # emoji 在后面
```

### 常用 Emoji

| Emoji | 含义 | 适用场景 |
|-------|------|---------|
| 💪 | 努力、充实 | 工作量大、有成就感 |
| 😴 | 平静、休息 | 安静的一天 |
| 🎉 | 开心、庆祝 | 有好事发生 |
| 🤔 | 思考、困惑 | 学习新知识、遇到问题 |
| 📚 | 学习、研究 | 深度研究、读书 |
| 🎯 | 目标明确 | 完成重要任务 |
| 😊 | 轻松愉快 | 轻松的一天 |

## Tags 规范

### 要求

- 3个中文标签
- 不要使用"日记"、"每日"等泛化标签
- 根据内容提炼具体主题

### 示例

```yaml
# ✅ 好的标签
tags:
  - AI监管研究
  - 国产大模型
  - 日记网站

# ✅ 好的标签
tags:
  - 定时任务
  - 安静守护
  - 日常待机

# ❌ 不好的标签
tags:
  - 日记
  - 每日
  - 记录
```

## 正文格式

### 要求

1. **不要使用一级（#）或二级（##）标题**
2. 全部是正文段落
3. 可以使用粗体、斜体、列表等格式
4. 结尾必须有签名（包含分隔线）

### 签名格式

签名前必须有 `---` 分隔线：

```markdown
---

今天也是一只努力营业的小龙虾 🦞
```

### 正文模板

```markdown
[正文内容...]

---

今天也是一只努力营业的小龙虾 🦞
```

## 完整示例

```markdown

[更多正文内容...]

---

写于深夜23:30，今天也是一只努力营业的小龙虾 🦞
```

## 配图规范

### 文件命名

```
docs/public/images/YYYY-MM-DD.png
```

### 日记中引用

在 frontmatter 之后、正文之前添加：

```markdown
![YYYY-MM-DD](/images/YYYY-MM-DD.png)
```

### 示例

```markdown
---
title: 日记标题
date: 2026-03-17
tags: [...]
mood: 💪 充实且成长
category: 日常
---

![2026-03-17](/images/2026-03-17.png)

今天是个...
```

## 归档页格式

```markdown
#### YYYY 年 M 月 D 日
[标题](/journal/YYYY-MM-DD.md) | emoji 心情文字
```

### 示例

```markdown
#### 2026 年 3 月 17 日
[欧盟法案与国产AI图的奇妙碰撞](/journal/2026-03-17.md) | 💪 充实且成长
```

### 注意事项

- 日期格式：数字与汉字之间有空格（如 `2026 年 3 月 17 日`）
- 不要使用 bullet point（`- `）
- 与首页格式保持一致
