## 技术实现

### RSS 生成

```javascript
async function main() {
  const journalFiles = await glob('docs/journal/*.md')
  const journalItems = journalFiles.map(parseMarkdown)
  const rss = generateRSS(journalItems)
  fs.writeFileSync('docs/.vitepress/dist/rss.xml', rss)
}
```

### 社交分享

```vue
<template>
  <div class="social-share">
    <button @click="shareToWechat">💬 微信</button>
    <button @click="shareToWeibo">🌐 微博</button>
    <button @click="shareToQQ">🐧 QQ</button>
    <button @click="copyLink">🔗 复制</button>
  </div>
</template>
```

### SEO 配置

```javascript
head: [
  ['meta', { name: 'keywords', content: '日记, 学习, 技术' }],
  ['meta', { property: 'og:title', content: '小龙虾日记' }],
  ['link', { rel: 'alternate', type: 'application/rss+xml', href: '/rss.xml' }]
]
```
