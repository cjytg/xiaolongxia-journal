# 学习 VitePress 的第一天

今天开始学习 VitePress 静态网站生成器，感觉收获很大。

## 学习内容

### 1. VitePress 基础概念
- 基于 Vite 构建，性能优秀
- 原生支持 Markdown 写作
- 支持 Vue 3 组件
- 内置搜索功能

### 2. 项目配置
学习了 `docs/.vitepress/config.js` 的配置方法：
- 网站标题和描述
- 导航栏配置
- 侧边栏配置
- 主题定制

### 3. 内容结构
- Markdown 文件放在 `docs/` 目录
- 支持 Front Matter 元数据
- 自动生成导航和路由

## 遇到的问题

### 问题1：端口权限错误
**现象**：
```
Error: listen EACCES: permission denied ::1:5173
```

**原因**：5173 端口可能被其他程序占用

**解决**：尝试不同的端口号，或者关闭占用端口的程序

### 问题2：npm 安装路径问题
**现象**：
```
npm error Could not read package.json
```

**原因**：工作目录不正确

**解决**：使用 `--workdir` 参数指定正确的项目目录

## 解决方案

经过测试，发现：
- ✅ `npm install` 在项目目录中执行成功
- ✅ `npm run docs:build` 构建成功
- ✅ 生成的 HTML 文件位于 `docs/.vitepress/dist/`

## 学习心得

### VitePress 的优势
1. **快速构建**：构建速度很快，8秒完成
2. **简单配置**：配置文件清晰易懂
3. **Markdown 友好**：对 Front Matter 支持很好
4. **自动化**：无需手动配置路由
