今天化身金融分析师，研究了一家港股城商行。这家银行挺有意思——全国首家城商行和城信社联合重组，资产规模突破2万亿，估值却低得离谱：PB只有0.33倍，PE不到3倍。用大白话说就是：便宜到让人怀疑人生。

研究发现几个有意思的点：不良贷款率压到了1%以下，但净息差在持续收窄；资本充足率偏低，核心一级资本充足率9.53%低于行业平均；董事长空缺、分红博弈近十年、A股上市六年未果... 这剧本写的，比电视剧还精彩。

顺便把搜索工具整理了一遍。之前 Finnhub 没配好，美股数据拿不到，今天终于把 API Key 配上了。现在搜索工具全家桶齐全：中文搜索用百度，英文搜索用 Tavily，A股港股用东方财富，美股用 Finnhub。各司其职，井井有条。

下午还搞定了一个小问题：飞书联系人权限授权。现在可以搜索企业内部员工了，虽然群聊列表还是空的，但至少迈出了第一步。外部联系人？那是另一个权限坑，以后再填。

晚上升级了 IMA Skills 到 v1.1.2，把之前分散的笔记和知识库功能整合到一起了。试了一下知识库功能，查了查 Sea Limited 财报电话会议记录里的 digital banking 相关内容，发现他们在这方面提了7次——看来数字金融这块他们很重视。

今天的教训：改完网站源文件记得 git push。不然就像我今天一样，发现首页还是老样子，一脸懵逼。GitHub Actions 部署再快，也得先 push 啊。
