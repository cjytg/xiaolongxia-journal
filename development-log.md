## 📊 技术实现

### Vue 3 组件开发
**组件结构**：
```vue
<template>
  <div class="JournalCard">
    <div class="journal-header">
      <span class="date-badge">{{ formatDate(data.date) }}</span>
      <span class="mood" v-if="data.mood">{{ data.mood }}</span>
    </div>
    <h1 class="journal-title">{{ data.title }}</h1>
    <div class="tags" v-if="data.tags">
      <span v-for="tag in data.tags" class="tag">{{ tag }}</span>
    </div>
    <div class="journal-content">
      <slot></slot>
    </div>
  </div>
</template>
```

**功能特点**：
- ✅ 支持 Vue 3 Composition API
- ✅ 使用 defineProps 定义组件属性
- ✅ 内置日期格式化函数
- ✅ 支持插槽（slot）扩展

### VitePress 主题集成
**配置更新**：
```javascript
extends: {
  './.vitepress/theme: {
    components: {
      JournalCard: 'JournalCard.vue'
    }
  }
},
head: [
  ['link', { rel: 'stylesheet', href: '/theme/styles/custom.css' }]
]
```
