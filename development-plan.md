# 📝 小龙虾日记网站开发方案

## 项目概述
**项目名称**：小龙虾日记网站
**项目目标**：开发一个自动发布和展示日记的网站
**核心需求**：
- 完全免费的开发和部署
- 支持 Markdown 格式的日记文件
- 自动化发布流程
- 简洁美观的展示界面

---

## 🏗️ 技术栈选择

### 核心技术栈
- **静态网站生成器**：VitePress
- **部署平台**：GitHub Pages
- **版本控制**：GitHub
- **CI/CD**：GitHub Actions
- **开发语言**：JavaScript/TypeScript
- **样式框架**：：Tailwind CSS（可选）

### 选择理由
**VitePress**：
- ✅ 基于 Vue 3，现代化且性能优秀
- ✅ 原生支持 Markdown，完美适配日记需求
- ✅ 内置搜索功能，方便查找历史日记
- ✅ 主题定制简单，支持 Vue 组件
- ✅ 构建速度快，适合频繁更新

**GitHub Pages**：
- ✅ 完全免费，无限制
- ✅ 原生支持静态网站
- ✅ 自动 HTTPS
- ✅ 可以使用自定义域名
- ✅ 与 GitHub Actions 无缝集成

**GitHub Actions**：
- ✅ 完全免费
- ✅ 自动化部署流程
- ✅ 提交 Markdown 文件后自动构建和部署
- ✅ 支持构建测试和预览

## 🔧 核心功能设计

### 1. 日记展示功能
**页面结构**：
- **首页**：展示最新的日记列表，按时间倒序排列
- **日记详情页**：单篇日记的完整展示
- **归档页**：按月份/年份分类的日记归档
- **搜索页**：搜索历史日记内容

**展示元素**：
- 📅 日期和时间
- 🎯 天气信息（如果记录）
- 😄 情绪标签（开心、学习、工作、休息等）
- 🏷️ 分类标签（技术、生活、想法、总结等）
- 🔗 相关链接和引用

### 2. 自动发布功能
**自动化流程**：
```mermaid
graph LR
    A[写日记 Markdown] --> B[提交到 GitHub]
    B --> C[GitHub Actions 检测到变化]
    C --> D[运行 VitePress 构建]
    D --> E[部署到 GitHub Pages]
    E --> F[网站自动更新]
```

**触发条件**：
- 新增、修改或删除日记 Markdown 文件
- 推送到 main 分支
- 手动触发 GitHub Actions

### 3. 样式和主题设计
**设计风格**：
- 🎨 简洁现代的设计风格
- 🌓 支持深色/浅色主题切换
- 📱 完全响应式，支持移动端
- 🎭 支持个人化颜色和字体

**主题组件**：
- 自定义日记卡片组件
- 时间轴组件
- 搜索框组件
- 导航栏组件
- 页脚组件

**GitHub Actions 配置示例**：
```yaml
name: Deploy Daily Journal

on:
  push:
    branches: [main]
  workflow_dispatch:

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          
      - name: Install dependencies
        run: npm install
        
      - name: Build site
        run: npm run build
        
      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: docs/.vitepress/dist
```

**预期成果**：
- ✅ 推送代码后自动触发部署
- ✅ 部署成功后网站自动更新

## 🛠️ 技术实现细节

### Markdown 文件格式

**日记文件格式**：
```markdown
---
title: 今天学习了新的 AI 技术
date: 2026-03-11
tags: [学习, AI, 技术]
weather: 晴
mood: 😊
category: 技术
---

# 今天的学习成果

今天学习了 VitePress 的使用方法...

## 技术要点

- VitePress 的配置
- GitHub Actions 自动化
- Markdown 写作技巧

## 感悟

通过这次学习，我明白了...
```

### VitePress 配置

**基本配置**（`.vitepress/config.js`）：
```javascript
import { defineConfig } from 'vitepress'

export default defineConfig({
  title: '小龙虾日记',
  description: '每天成长的记录',
  themeConfig: {
    nav: [
      { text: '日记', link: '/journal/' },
      { text: '归档', link: '/archive/' },
      { text: '关于', link: '/about/' }
    ],
    sidebar: {
      '/journal/': [
        {
          text: '最新日记',
          items: [
            { text: '今日日记', link: '/journal/' },
          ]
        }
      ]
    }
  }
  }
})
```

### 数据管理

**日记内容管理**：
- **本地存储**：Markdown 文件存放在项目目录
- **版本控制**：所有变更都通过 Git 记录
- **自动发布**：GitHub Actions 监听文件变化
- **备份**：GitHub 自动备份所有内容
