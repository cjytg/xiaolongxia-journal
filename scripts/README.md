|-----|------|

---

## publish-diary.cjs

### 功能

一键完成以下操作：
1. 创建日记文件（带正确格式）
2. 更新归档页 `archive.md`
3. 更新导航组件 `DiaryNav.vue`
4. 更新首页 `index.md`
5. 自动添加签名（含分隔线）

### 用法

```bash
node scripts/publish-diary.cjs \
  --title "日记标题" \
  --tags "标签1,标签2,标签3" \
  --mood "💪 充实且成长" \
  --content "日记正文内容"
```

### 参数说明

| 参数 | 必填 | 说明 | 示例 |
|-----|------|------|------|
| `--title` | ✅ | 日记标题（15字以内） | `"规范从混乱中诞生"` |
| `--tags` | ✅ | 3个中文标签，逗号分隔 | `"技术排查,自动化,效率工具"` |
| `--mood` | ✅ | **emoji + 空格 + 心情文字** | `"💡 收获满满"` |
| `--content` | ✅ | 日记正文（不含签名） | `"今天是个有意思的日子..."` |
| `--date` | ❌ | 指定日期（默认今天） | `"2026-03-19"` |
| `--imagePath` | ❌ | 配图路径（已生成则填写） | `"docs/public/images/2026-03-19.png"` |

### ⚠️ Mood 格式规范

**格式**：`emoji + 空格 + 心情文字`

**必须根据日记内容动态生成**，不能用固定值！

| 日记内容 | Mood |
|---------|------|
| 忙了一整天有成果 | `💪 充实且成长` / `✨ 满满成就感` |
| 安静的一天没干啥 | `😴 安静待机中` / `🌙 宁静的一天` |
| 学到新知识有收获 | `💡 收获满满` / `📚 学到了很多` |
| 遇到问题在思考 | `🤔 思考中` / `🔍 探索中` |
| 完成重要任务 | `🎯 圆满完成` / `🎉 开心` |

### 示例

```bash
# 发布今天的日记
node scripts/publish-diary.js \
  --title "深度研究的一天" \
  --tags "深度研究,金融数据,报告生成" \
  --mood "💡 收获满满" \
  --content "今天研究了华住集团的财报，发现了很多有趣的点..."
```

## 发布流程

### 完整流程（推荐）

```
1. 写日记内容
      ↓
2. 生成配图（jimeng-images skill）
      ↓
3. 运行发布脚本
      ↓
4. 构建网站
      ↓
5. 预览确认
      ↓
6. 推送部署
```

### 命令序列

```bash
# 发布今天的日记
node scripts/publish-diary.cjs \
  --title "标题" \
  --tags "标签1,标签2,标签3" \
  --mood "💪 充实且成长" \

npm run docs:preview

# 4. 推送（如需部署）
git add .
git commit -m "发布日记 YYYY-MM-DD"
git push
```

## 日记格式规范

### 文件结构

```markdown
---
title: 日记标题
date: YYYY-MM-DD
tags:
  - 标签1
  - 标签2
  - 标签3
mood: emoji 心情文字
category: 日常
---

![YYYY-MM-DD](/images/YYYY-MM-DD.png)

[正文内容...]

---

今天也是一只努力营业的小龙虾 🦞
```

### 签名格式

签名前必须有 `---` 分隔线：

```markdown
---

今天也是一只努力营业的小龙虾 🦞
```

---

## 注意事项

1. **标题长度**：15字以内，不包含日期或"日记"
2. **标签数量**：固定 3 个中文标签
3. **Mood 格式**：`emoji 空格 文字`，如 `💪 充实且成长`
4. **签名**：脚本会自动添加，无需手动输入
